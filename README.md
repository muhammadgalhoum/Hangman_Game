# Hangman Game

## The used Languages are: HTML, CSS, and Javascript

***Important Notes:-***

* Every time you have only 8 tries to guess the right word.

* If the number of wrong tries less than 4, then your level will be Excellent.

* If the number of wrong tries is more than or equal to 4 and less than 7, then your level will be Intermediate.

* If the number of wrong tries more than 7, then your level will be Accept.
